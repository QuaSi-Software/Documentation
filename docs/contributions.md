# Contributions
We welcome contributions to our open source tools and the overall project QuaSi! We wish to bring the results of the time and resources invested in this project to the public and would like to see the tools used, developed and adapted to many different purposes.

If you want to contribute to the project, there are many different ways to do so:

* Create a bug report for something you noticed
* Request a feature / change for something that the tools are missing
* Let us know when you use the tools in your work
* Fix bugs you or others have reported
* Update the documentation where it is incorrect or out of date
* Help develop missing features
* Discuss cooperation on future projects that might benefit from the tools
* And many more

For anything that affects the tools' code or documentation, you can use the issue tracker of the tools [on GitHub](https://github.com/orgs/QuaSi-Software/repositories) to communicate. For more general feedback you can also write us at [info@quasi-software.org](mailto:info@quasi-software.org).
